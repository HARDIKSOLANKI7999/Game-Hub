# 🎮 Game Hub

A modern and responsive gaming discovery platform built with Angular 21.  
Browse popular games, explore categories, search titles, and enjoy a clean UI inspired by modern gaming platforms.

🌐 **Live Demo:**  
https://hardiksolanki7999.github.io/Game-Hub/

---

## 🚀 Features

- 🔍 Search games instantly
- 🎮 Browse trending and popular games
- 🧩 Filter by genres and platforms
- 📱 Fully responsive design
- ⚡ Fast and optimized Angular application
- 🎨 Clean and modern UI
- 🌙 Smooth user experience

---

## 🛠️ Tech Stack

- **Framework:** Angular 21
- **Language:** TypeScript
- **Styling:** CSS / SCSS
- **Build Tool:** Angular CLI
- **Testing:** Vitest
- **Deployment:** GitHub Pages

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/hardiksolanki7999/Game-Hub.git
```

# Navigate to the project directory:

```bash
cd Game-Hub
```

# Install dependencies:

```bash 
npm install
```

# Start the development server:

```bash 
ng serve
```

# Open your browser and visit:

```bash
http://localhost:4200/
```
